# 龚海滨的学习管理系统

一个简洁的个人学习追踪平台，支持学习记录录入、日历管理、任务跟踪等功能。

## 功能

- 📝 **学习录入**：手动录入/语音录入，支持图书、视频、音频文稿三种类型
- 📅 **日历管理**：可视化学习强度，按日期查看记录
- ✅ **任务管理**：待办任务跟踪
- 📚 **学习档案**：按内容/日期查看学习记录
- 🎯 **答题考核**：录音作答，1-10星评分
- 📊 **数据统计**：学习时长统计图表

## 技术栈

- 纯前端 SPA（HTML + CSS + JavaScript）
- 响应式设计（桌面端 + 移动端）
- localStorage 本地存储

## 部署到 Vercel

1. Fork 本仓库
2. 访问 [Vercel](https://vercel.com)
3. 导入 GitHub 仓库 `Nathan-Hg/gonghaibin-learningmanagement`
4. 点击 Deploy
5. 获得永久免费的公网访问地址

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/Nathan-Hg/gonghaibin-learningmanagement.git

# 进入目录
cd gonghaibin-learningmanagement

# 启动本地服务器（任选一种）
python -m http.server 8080
# 或
npx serve .
```

然后访问 http://localhost:8080
